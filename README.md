# 🧩 Mini Dashboard

Un dashboard personnel modulaire construit avec **Angular 21** et **TypeScript** — conçu pour centraliser les infos utiles du quotidien en un seul coup d'œil.

🔗 **[Voir la démo live →](https://dashboard-pp-nine.vercel.app)**

---

## ✨ Fonctionnalités actuelles

| Widget | Description |
|--------|-------------|
| 🌤 **Météo** | Météo en temps réel via API géolocalisée |
| ✅ **Tâches du jour** | Liste de tâches simple et persistante |
| 💬 **Phrase du jour** | Citation aléatoire pour bien démarrer la journée |
| 🔲 **Widget 4** | *En cours — à venir* |
| 🔲 **Widget 5** | *En cours — à venir* |

---

## 🛠 Stack technique

- **Framework** : Angular 21
- **Langage** : TypeScript
- **Style** : SCSS
- **Déploiement** : Vercel
- **APIs** : Géolocalisation + météo (Open-Meteo / OpenWeatherMap)

---

## 🚀 Lancer le projet en local

```bash
# Cloner le repo
git clone https://github.com/ItsLok0/Mini-Dashboard.git
cd Mini-Dashboard

# Installer les dépendances
npm install

# Lancer le serveur de dev
ng serve
```

Ouvre ensuite [http://localhost:4200](http://localhost:4200) dans ton navigateur.

## 💡 Pourquoi ce projet ?

L'objectif était de construire une vraie application Angular en autonomie — avec gestion d'APIs externes, architecture modulaire par composants, et déploiement en production. Un terrain de jeu pour pratiquer les bonnes pratiques Angular hors contexte professionnel.
